# Introduction
This repo is a Pytorch implementation of **Dual Information Enhanced Multi-view Attrubuted Graph Clustering**, [arxiv](https://arxiv.org/abs/2211.14987).

# Requirements

```
torch = 1.10.1+cu111
python = 3.6.13
```

# Run

Training and evaluation.

```run
python run.py
```
