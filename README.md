# Introduction
This repo is a Pytorch implementation of **Dual Information Enhanced Multi-view Attrubuted Graph Clustering**, IEEE Transactions on Neural Networks and Learning Systems, 2025.

# Requirements

```
torch = 1.10.1+cu111
python = 3.6.13
```

# Run

Training and evaluation.

```run
python run.py
```
